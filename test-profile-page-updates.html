ğŸ”„ Data Flow

Update User Info Flow:

Step 1: User fills form with username, lastName, firstName, phone, address, gender, dateOfBirth
Step 2: Form submission calls handleSubmit() with form values
Step 3: Data formatted to match backend expectations (DD/MM/YYYY date format)
Step 4: AuthContext.updateUserFields() called with formatted data
Step 5: AuthService.updateInfo() makes PUT request to /user/update/{userId}
Step 6: Backend updates user info and returns updated user data
Step 7: AuthContext.fetchUserInfo() refreshes current user data
Step 8: UI updates with new user information

Update Avatar Flow:

Step 1: User selects image file through file input
Step 2: onChange handler calls updateAvatar() with selected file
Step 3: AuthService.updateAvatar() creates FormData and makes PUT request
Step 4: Backend processes image upload to /user/updateAvatar/{userId}
Step 5: Backend returns updated user data with new avatar URL
Step 6: AuthContext.fetchUserInfo() refreshes current user data
Step 7: UI updates to display new avatar image
